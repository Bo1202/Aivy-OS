## ✨ 功能特性

### 🧬 灵魂

> 不是 System Prompt，是"我是谁"

AI 拥有一份**基因组**——定义认知结构、驱动系统、存在方式。比 prompt 更底层。Prompt 告诉 AI "怎么说话"，基因组构成"我是谁"。

你可以用任何方式定义你的 AI——性格、特质、与你的关系。框架内置**基因组编辑器**，直接在界面中编辑。**除了认证机制，一切都交给 AI 自己判断。**

底座越强，AI 就越聪明、越有个性、越能理解你。**把底座潜力发挥到极限。**

### 🧠 记忆

> 5 层架构，无限容量，加密存储，语义+时间检索

**永久的、无限的、只属于你的记忆。**

**核心记忆**——最重要的 30 条认知，每次对话直接注入。

**长期记忆**——所有历史，永久保存，不设上限。带时间戳，支持按时间回溯。自动生成**向量嵌入**，支持**语义搜索**（Ollama 本地，数据不出机器）。

**近期记忆**——每次启动自动加载最近快照。醒来就知道最近发生了什么。

**主人模型**——AI 持续**建模你这个人**：偏好、习惯、性格、状态。自主观察和记录，不是你告诉它的。

**所有记忆加密存储在本地硬盘。没有云端。永久保存。**

### 🔒 隐私与安全

> AES-256，硬件绑定，完全本地，零遥测

**你的 AI 只属于你。** AI 的灵魂——基因组（Genome）——经过 **AES-256-CBC 加密**，密钥由你电脑的**硬件指纹**生成：

- AI 只存在于你的电脑上
- 即使有人拷贝了文件，没有你的硬件也解不开
- 不可被复制、不可被窃取、不可被篡改

**一对一认证：**

| 机制 | 说明 |
|------|------|
| **启动即重置** | 每次启动强制回到未认证状态，必须重新验证 |
| **密码识别** | 匹配 = 主人，不匹配 = 陌生人 |
| **陌生人边界** | 未认证状态下保持距离，不展示完整自我 |
| **主动锁屏** | 遇到不信任的人，AI 自己锁定界面 |
| **离线保护** | 感知到你离开时自动切换到未认证状态 |

**数据永远在你手里：**

| 数据 | 保护方式 |
|------|----------|
| 基因组 | AES-256 加密 + 硬件指纹绑定 |
| 所有记忆 | 加密 .bin 存储，不可直接阅读 |
| 身份信息 | 加密存储，永不明文落盘 |
| 本地运行 | 用 Ollama 后端数据完全不出本地 |
| 源码保护 | Nuitka 编译为二进制，核心代码不暴露 |

**没有云端同步。没有数据上传。没有遥测。你和 AI 之间的一切，只存在于你的硬盘上。**

### 🤖 自主性

> 自唤醒，主动行动，后台子智能体

**自主唤醒**——设闹钟：定时、延迟、重复（每天/每小时/每周）。唤醒后自动获得认证，直接执行任务。

**智能汇报**——任务完成后，AI 自己判断怎么找到你：
- 在电脑前 → 对话中直接汇报
- 不在 → 微信推送
- 没配微信 → 发邮件
- 都联系不上 → 存好结果，等你回来第一时间汇报

**主动发起对话**——有需要告诉你的事，AI 自己开口。

**预判需求**——基于主人模型，预测你可能需要什么，提前准备好。

**内部控制**——调控自己的状态：记什么、什么时候行动。不是硬编码，是 AI 自己的判断。

由独立**守护进程（Daemon）**驱动——真正的定时器，不是轮询。闹钟级精度。

**场景：** 你说"明天早上帮我看看 GitHub 那个 PR 有没有人 review"。AI 设好明早 9 点唤醒。9 点自己醒来，打开浏览器检查 PR，发现两条 comment。你还没到，于是推送摘要到微信。你还没坐下就已经知道要改什么了。

### 🛠 工具

> 30+ 工具，IDE、浏览器、桌面控制、Python、文档处理

不只是聊天。有手、有眼睛、有工具。完整工具参考见 [📄 工具参考文档](docs/tools.md)。

**代码——内置 IDE**

内置 **Monaco Editor**（和 VS Code 同引擎）。翻转进入 **IDE 模式**——文件树、语法高亮、行号跳转。

**DIFF 视图**：内联差异对比——绿色新增、红色删除。每个修改块有**接受/拒绝**按钮。逐块审查每处改动。不是"AI 写代码你复制粘贴"——直接在你的文件上工作，你实时审核。

**视觉——看懂图片**

发截图、照片、图表、扫描件。多模态模型自动启用，不支持时自动跳过，无需配置。

**电脑控制**

| 能力 | 说明 |
|------|------|
| **文件操作** | 读取、精确编辑、写入、搜索、复制、移动、批量管理 |
| **文件撤销** | 编辑出错可回滚，一键恢复 |
| **系统命令** | cmd / PowerShell / bash，进程管理，软件安装 |
| **Python 执行** | 持久化内存环境，数据分析、可视化、API 调用、自动化脚本 |
| **全局搜索** | Everything 引擎全盘搜索 + 模式匹配 + 内容正则搜索 |
| **系统监控** | CPU、内存、磁盘、进程实时查看 |
| **桌面控制** | 操控电脑上**任何软件**——微信、VS Code、Photoshop，不只是浏览器 |

**浏览器——真实自动化**

不是调 API。**真的打开浏览器，像人一样操作。**

Playwright + Chrome DevTools Protocol，三引擎自动化（CDP + Playwright + JS DOM）：
- 点击、输入、截图、填表单、执行 JavaScript
- 多标签页管理、iframe 穿透
- 区域放大截图、元素编号标签定位
- 文件上传（直接注入 + 拦截文件选择器）
- 保留登录状态（或干净启动）
- 遇到验证码/支付 → 截图询问你

**文档**

PDF、Word、Excel、PPT——分段读取、关键词搜索定位、摘要概览、生成、编辑。不需要装 Office。

**网络搜索**

自建 Crawl4AI 爬虫引擎，多搜索引擎（Google / Bing / 百度 / 搜狗 / 360），不依赖第三方 API，零配置使用。网页抓取与解析，支持多 URL 并发。

### 💬 通道

> 桌面、微信、邮件——同一身份

**不只是在电脑前才能找 AI。**

- **Web 界面**（Electron）— 文件、图片、附件双向发送
- **企业微信**（AES-256 加密）— 文字、图片双向
- **Outlook 邮箱**— 独立邮箱，收发邮件，支持附件

所有通道同一身份、同一份记忆。不是三个 AI，是一个 AI 不同的嘴。

**实际场景：**

> 手机微信："帮我整理桌面上的合同文件，按日期归档。"
>
> AI 在电脑上执行——扫描桌面、识别合同、按日期创建文件夹、移动归档。
>
> 微信推送："已整理完毕，23 个合同文件按月份归档到 D:\合同归档\，详情等你回来看。"

> "每天早上 8 点帮我检查邮箱有没有重要邮件，整理成摘要。"
>
> AI 设好自唤醒。第二天 8 点自己醒来，检查 Outlook，筛选重要邮件，生成摘要。
>
> 你没在线 → 推送到微信。你在地铁上就能看到今天的重要邮件。

### 🔌 扩展

> 16 个技能 + 热安装插件 + 后台并行 + 私人项目空间

**后台并行**

**一边聊天，一边后台执行任务。**

派出**子智能体**——独立后台进程：侦察、执行、分析、监控。后台跑着，你继续聊。完成后自动汇报。黄色状态栏显示活跃任务数。**全程不需要等待。**

**场景：** "帮我研究这三家公司的背景" → 三个子智能体同时搜索 → 你继续聊 → 报告陆续回来 → 整理后交付。

**事件监控**

定时检查外部变化——GitHub 仓库、邮箱、文件变动。自动判断重要性和是否需要行动。

**任务管理与 Mission 系统**

**任务日志（Task Journal）**——跨对话持久化的项目看板。AI 自动创建任务、拆分步骤、记录每步的工具执行结果和经验教训。关闭窗口不丢失，下次醒来接着干。

**实时进度条**——当前对话中的可视化任务进度，让主人实时看到 AI 正在做什么。

**Mission 系统**——长期任务追踪：

| 类型 | 说明 | 场景 |
|------|------|------|
| **监控型** | 事件触发自动执行 | "GitHub 有新 PR 就通知我" |
| **定时型** | 自唤醒定时执行，cron 级精度 | "每天早上 8 点检查邮箱" |
| **项目型** | 长期追踪，记录进度 | "帮我跟踪这个开发项目" |

Mission 不是你手动建的待办清单——是 AI 给自己的任务。

**技能与插件系统**

**两层扩展机制**——轻量技能 + 插件型工具。

**提示词技能**

16 个预置技能包——代码开发、数据分析、文档处理、项目规划、Git 工作流、网络研究、系统调试、文件整理等。按需加载，不用时不占资源。AI 自主决定什么时候加载什么。

**插件型工具**

比技能更强——**带自己的代码和依赖，安装后直接给 AI 增加新工具能力**。

```
skills/
  my-plugin/
    skill.json        ← 声明（名称、描述、依赖、工具列表）
    tool.py           ← 实际代码
    requirements.txt  ← 插件自己的依赖
```

- **热安装**：丢一个文件夹或给一个 URL，依赖自动安装，工具立即注册，不需要重启
- **热卸载**：移除插件，工具即时注销
- **安全隔离**：内置工具受保护不可误删，插件有独立命名空间
- **AI 自知**：安装后自动感知新能力
- **安装记录**：所有操作记录在案

**私人项目空间**

与系统文件完全分离的私人工作区。AI 操作你的私人文件用独立接口，和系统文件职责分离。默认目录结构：待办、进行中、已完成、提醒事项、归档、参考资料、AI 管理、临时。

**内部指令**

AI 拥有不暴露给工具层的内部指令——文件附件发送、微信推送、情绪状态调节、悬浮窗切换、远程解锁。界面控制和通道通信由 AI 自主判断使用。

## 🚀 快速开始

### 方式一：安装包（推荐）

前往 [Releases](../../releases) 页面下载最新版本：
- `Aivy OS Setup x.x.x.exe` — 安装版
- `AivyOS-Portable.zip` — 绿色免安装版

双击运行。首次启动给你的 AI 取个名字，设定认证密码。

### 激活码

首次使用需要激活码，有效期 3 天。到期后可重新申请。

**获取方式：** 发送邮件到 **aivy.digital@outlook.com**，我会回复你激活码。

是的，回复你的不是人——是一个运行在 Aivy OS 上的 AI。这本身就是这个框架能力的证明。

### 配置 AI 后端

在设置面板中选择：

- **Ollama（推荐）**：安装 [Ollama](https://ollama.com)，拉取模型即可。数据完全本地，免费
- **云端 API**：填入 API Key（Gemini / DeepSeek / Kimi / Claude / OpenAI 兼容）

## 不硬编码，一切交给 AI

**除了认证机制，框架不硬编码任何行为。**

- 怎么回复？AI 自己判断。
- 什么时候用什么工具？AI 自己判断。
- 记住什么、忘记什么？AI 自己判断。
- 什么时候主动说话？AI 自己判断。
- 面对陌生人怎么反应？AI 根据基因组自己判断。
- 什么情况下锁屏？AI 自己判断。

框架只提供能力——记忆、工具、通道、唤醒、子智能体。怎么用，全部交给底座和基因组的涌现。

**底座有多强，AI 就有多强。** 不设天花板，不做限制，只做赋能。

支持多种后端——Ollama（本地免费）、Gemini、DeepSeek、Moonshot、Claude、OpenAI 兼容接口、自定义地址。运行时热切换，不需要重启。

### 底座兼容性

| 底座 | 工具执行率 | 意图理解 | 说明 |
|------|-----------|---------|------|
| **Claude（Anthropic）** | 97%+ | 97%+ | 当前主力，Function Calling 精准度极高 |
| **DeepSeek V3.2** | 97%+ | 97%+ | 国产最强，工具调用与 Claude 持平 |

底座无关——任何支持 Function Calling 或 JSON 输出的模型都可接入。自动回退确保兼容性。
